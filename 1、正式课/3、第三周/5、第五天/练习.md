可以利用递归去做

1、把以下对象的属性名放到一个数组里
```
    var obj = {
        a: {
            b:1,
            c:{d:1},
            e:2
        },
        m:0
    };
```





2、把以下对象的属性名放到一个数组里
```
    var obj = {
        n: {
            x:1,
            a: [
                {b: 1, c: [{d: 1}, {e: 1}]},
                {f: 1},3
            ]
        },
        m:0
    };
```
